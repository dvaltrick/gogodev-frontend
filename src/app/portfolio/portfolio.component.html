LaisApp

WEB APP desenvolvido para melhorar o sistema de compilação de uma software house. Nesse programa os testadores de software podem escolher o módulo que desejam compilar, após isso é enviada uma requisição para o servidor que realiza a compilação do código fonte e entrega o link para download do módulo compilado

Frontend: Agular2. Backend: GO Lang + .bat Script

Espaço do Cliente

Ferramenta de gerenciamento de tarefas, criado para que nossos clientes possam acompanhar o andamento dos seus projetos, dar sugestões e tirar dúvidas.

Frontend: Angular5 Backend: PHP Laravel
